# Listing Pages

Endpoint: `https://api.youcan.shop/pages`

Method: `GET`

## Sorting

- `sort field`:
  - `created_at`: Page Creation Date
  - `visibility`: Page Visibility
  - `name`: Page Name
  
  
## Searching
To search in categories, add GET param `q`.

`https://api.youcan.shop/pages?q=<SEARCH_VALUE>`

### Searcheable fields
Searchable fields are matched when the field value contains the expression

- `name`
- `slug`
- `content`
